## introduction

无限滚动列表，典型场景，比如天猫/京东浏览商品列表、今日头条客户端

难点：

| 功能          | 难度系数 | 描述                                                                  |
| ------------- | -------- | --------------------------------------------------------------------- |
| 动态新建/销毁 | ★★☆☆☆    | virtualizedList 思路，另外加上防抖动，比如在一个阈值附近来回切换      |
| 下拉更新      | ★★★★★    | 看似简单，但实际要模拟源生的体验很难                                  |
| 触底刷新      | ★☆☆☆☆    | 比较简单，触底判定在一些场景需要兼容，比如存在小数时，无法 === 来判定 |
